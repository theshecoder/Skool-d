OUR STORY

Skool'd is an application created for a capstone project in computer science at Brigham Young University. This application was created using Xamarin, a multi-platform langauge. This application allows for the planning of all assignments and exams in conjuction with the learning management system, Learnng Suite, at BYU. In addition, Skool'd provides time estimates and difficulty ratings for assignments in each particular class and provides a live feed of BYU's testing center. Skool'd is currently available on Android and IOS devices.
